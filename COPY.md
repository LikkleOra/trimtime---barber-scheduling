## 12. Voice & Tone Guidelines

### Brand Voice
- **Confident** - We know we're the best
- **Modern** - Urban, contemporary, fresh
- **Approachable** - Professional but not stuffy
- **Energetic** - Young, vibrant, dynamic

### Tone Characteristics
- Use active voice
- Short, punchy sentences
- Urban slang where appropriate ("legendary," "vibe the vibe")
- Emoji use for WhatsApp messages (✂️🔥📍⭐)
- ALL CAPS for emphasis on headings
- Avoid excessive formality

### Examples

✅ **Good:**
```
"Book A Fade"
"Vibe the vibe!"
"See the Legend's Gallery"
"Master Class Barbering"
```

❌ **Avoid:**
```
"Schedule an Appointment"
"We look forward to serving you"
"View our portfolio"
"Professional hair styling services"
```

## Notes for Copywriters

1. **Consistency:** Always use "FADEZONE" in all caps when referring to the brand
2. **Pricing:** All prices are in South African Rand (R)
3. **Time Format:** Use 12-hour format for customer-facing (2:30 PM), 24-hour for system (14:30)
4. **Date Format:** Full format for display (Friday, May 15, 2026), ISO format for database (2026-05-15)
5. **Phone Numbers:** Display format: 081 268 7806, System format: 27812687806
6. **Emoji Usage:** Primarily in WhatsApp messages and informal contexts
7. **Capitalization:** Service names in ALL CAPS (e.g., HAIR CUT, CHISKOP)
